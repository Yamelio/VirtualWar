Aide à la prise en main : 
	Avoir la liste des robots dans une base : Passer la souris sur une base, la liste s'affiche dans une bulle d'aide.
	Sélectionner un robot dans une base : Cliquer sur la base, chaque clic sélectionne le robot suivant dans la liste.
	Attaquer : Clic gauche sur votre robot, sélectionnez le bouton "Attaquer", et clic droit sur la cible.
	Se déplacer : Clic gauche sur votre robot, sélectionnez le bouton "Mouvoir", et clic (droit ou gauche) sur la cible. Si votre robot est un char et se déplace sur un robot (pour arriver sur la case entre eux deux), utiliser le clic droit pour sélectionner la case.

	/!\ La sélection d'une case se fait en cliquant sur le losange qui la représente, pas sur l'image qu'elle contient.
